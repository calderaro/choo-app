

# chootest
===============
testing the awesome choo library with babel, webpack, css-modules and hmr.

## Installation

### Clone
```bash
git clone git@gitlab.com:calderaro941/pxo-webclient.git
```
### Install dependencies
```bash
npm install
```
### Run development server
```bash
npm start
```
### Build for production
```bash
npm run build
```
