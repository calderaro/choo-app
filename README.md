

# Choo app
===============
Testing the awesome choo library with babel, webpack, css-modules and hmr.

## Installation

### Clone
```bash
git clone git@github.com:calderaro/choo-app.git
```
### Install dependencies
```bash
npm install
```
### Run development server
```bash
node index
```
Or
```bash
npm start
```
### Build for production (not available now)
```bash
npm run build
```
